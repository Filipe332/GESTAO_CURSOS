# Instruções para Instalação do Sistema de Gestão de Cursos

## Passos para Executar:

1. Extraia a pasta `gestao_cursos` em `C:\xampp\htdocs\`.
2. Abra o phpMyAdmin: http://localhost/phpmyadmin/
3. Crie um banco de dados chamado `sistema_gestao_cursos`.
4. Importe o arquivo `sql/gestao_cursos.sql` para o banco de dados.
5. Verifique o arquivo `conexao.php` para garantir que as configurações estão corretas.
6. Acesse o sistema pelo navegador: http://localhost/gestao_cursos/index.php
